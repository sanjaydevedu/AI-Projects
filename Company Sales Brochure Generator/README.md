# Company Sales Brochure Generator
## Overview

The Company Sales Brochure Generator is an AI-powered application that automatically creates professional, well-structured brochures for businesses. These brochures can be used for prospective clients, investors, and potential recruits, reducing the manual effort required to craft compelling company narratives.

The system takes a company name and its primary website URL as input and generates a polished, structured brochure in real time.

## 🎯 Business Problem

Creating high-quality sales brochures is time-consuming and requires a mix of content writing, branding, and structuring skills. Many companies struggle to quickly produce consistent and engaging materials for different stakeholders.

This project solves that problem by:

Automating brochure creation using AI
Ensuring consistent tone and structure
Reducing time from hours to seconds

## 💡 Solution

This project leverages Large Language Models (LLMs) to extract and generate meaningful business insights and present them in a structured brochure format.


## 🛠️ Tech Stack
### AI Models:
gpt-5-nano (fast, cost-efficient generation), 
gpt-4.1-mini (higher-quality content refinement)
### Backend:
Python, 
OpenAI API
### Other Tools:
Web scraping / content extraction from company websites, 
Prompt engineering for structured outputs

## ⚙️ How It Works
User inputs:
Company Name 
Website URL </br>
System:
Scrapes or processes website content
Sends structured prompts to LLM
Generates brochure sections </br>
Output:
Real-time streamed brochure
Clean, formatted content ready for use
